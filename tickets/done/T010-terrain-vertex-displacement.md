# T010: Terrain vertex displacement
**Priority:** P0 | **Phase:** 2 | **Size:** L
**Depends on:** T009

## Description
Apply the parabolic height function to displace vertices of a subdivided PlaneGeometry. Recompute normals after displacement.

## Acceptance Criteria
- [ ] Vertices displaced by height function
- [ ] Normals recomputed for correct lighting
- [ ] Visible hills and valleys in the scene
