# T025: Banking turns from roll input
**Priority:** P0 | **Phase:** 3 | **Size:** M
**Depends on:** T021

## Description
Roll input tilts the bird and induces a banking turn. The yaw rate is proportional to the roll angle, giving natural-feeling curved flight paths.

## Acceptance Criteria
- [x] Roll input banks the bird model
- [x] Banking induces proportional yaw rotation
- [x] Turn radius feels natural at flight speed
