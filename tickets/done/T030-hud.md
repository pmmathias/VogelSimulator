# T030: HUD altitude, speed, heading, flap indicator
**Priority:** P1 | **Phase:** 3 | **Size:** M
**Depends on:** T021

## Description
An HTML/CSS heads-up display overlaying the 3D scene. Shows current altitude, airspeed, compass heading, and a flap indicator that lights up when the player flaps.

## Acceptance Criteria
- [x] Altitude displayed in meters
- [x] Speed displayed in m/s or km/h
- [x] Compass heading displayed in degrees
- [x] Flap indicator activates on flap input
