# T026: Pitch control dive/climb
**Priority:** P0 | **Phase:** 3 | **Size:** M
**Depends on:** T021

## Description
Pitch input tilts the bird nose-up or nose-down. Pitching down increases speed at the cost of altitude; pitching up trades speed for altitude gain.

## Acceptance Criteria
- [x] Pitch input rotates bird around lateral axis
- [x] Nose-down pitch increases speed and descent rate
- [x] Nose-up pitch trades speed for climb
