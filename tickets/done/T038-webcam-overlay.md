# T038: WebcamOverlay PiP with skeleton drawing
**Priority:** P1 | **Phase:** 4 | **Size:** M
**Depends on:** T031, T032

## Description
A picture-in-picture overlay in the corner of the screen showing the webcam feed with the detected pose skeleton drawn on top. Helps the player see their tracked body position.

## Acceptance Criteria
- [ ] Webcam feed displayed in corner overlay
- [ ] Pose skeleton lines drawn over video
- [ ] Overlay can be toggled on/off
- [ ] Does not impact main render performance significantly
