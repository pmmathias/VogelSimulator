# T039: Calibration flow raise arms to calibrate
**Priority:** P1 | **Phase:** 4 | **Size:** L
**Depends on:** T032, T033

## Description
A calibration step before gameplay where the player raises both arms to set neutral position. This accounts for varying body proportions and camera angles, establishing baseline values for arm angles.

## Acceptance Criteria
- [ ] Calibration screen prompts user to raise arms
- [ ] Neutral arm angles recorded as baseline
- [ ] Subsequent input calculated relative to calibration
- [ ] Recalibration can be triggered during gameplay
